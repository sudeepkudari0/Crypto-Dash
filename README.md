## Run Locally

Clone the project

```bash
  git clone https://github.com/sudeepkudari0/Crypto-Dash.git
```

Go to the project directory

```bash
  cd Crypto-Dash
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
